In Sass, variables start with a $ followed by the variable name. $main-fonts: Arial, sans-serif; $headings-color: green; //To use variables: h1 { font-family: $main-fonts; color: $headings-color; } One example where variables are useful is when a number of elements need to be the same color. If that color is changed, the only place to edit the code is the variable value. */
